锁定

锁定模式

锁定有两种模式；一般使用默认的HOLD模式，持续按住锁定，松开取消锁定；另一种CLICK模式，按一下锁定，再按一下取消锁定

自动锁定

当人物离敌人很近时自动锁定功能就会启动，有个淡蓝色的锁定圈，这时与自己主动使用锁定的状态是一样的，所以此时无法使用变向蓄力斩

锁定距离

靠敌人一定距离内才能启动锁定，太远是不能启动锁定功能的，但如果在近处启动锁定，再以锁定的状态移动到锁定启动范围之外，锁定状态是不会被强制取消的

锁定切换

这个功能多用于蓄力斩中，但有一点要注意，锁定切换不是瞬时的，锁定圈从一个目标转移到另一个目标上是要一小段时间的，在这一小段时间内没有锁定圈，也没有锁定目标，如果在这段时间放出蓄斩将属于变向蓄力斩范畴

若在锁定状态移动到所有敌兵锁定距离以外，此时按下锁定切换键将失去锁定效果

取消锁定

先锁定一个目标后，再松开锁定，此时在无锁定状态下再出标已不是原来的人物正前方了，而是刚才锁定目标的方向
