难度

难度共有 EASY NORMAL HARD SUPER 四种

EASY NORMAL 两个难度的吸魂速度一样(较慢)

HARD SUPER 两个难度的吸魂速度一样(较快)

难度不影响吸阳魂速度

难度不影响杀阵连接时间

难度不影响杀阵倍率

难度不影响蓄斩倍率

难度不影响敌方三面倍率

难度越高敌方AI越高，敌方攻击力越高

难度对刀的攻击力有影响

难度对秀真和守恒的手里剑攻击力无影响

难度对乔武藏的手里剑攻击力有影响

难度对忍术的攻击力无影响(有特殊情况)

各难度下敌方HP不变化，为一固定值(有特殊情况)

难度对部分兵死后所出魂量也有所影响，比如最终BOSS战灵符，低难度出一个魂，高难度出两个魂

日版、美版、欧版在同等难度中，数据方面没有差别
