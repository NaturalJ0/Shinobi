手里剑

单发

单发手里剑有各种动作，每种动作威力都相同，威力不受杀阵影响；秀真和守恒的手里剑除攻击力外还附带定身麻痹效果，时间维持较短，敌人受到攻击(包括足踢)或死亡都会中断效果；在墙上DASH后人物未离开墙，此时出标方向与上一次DASH方向相同

八双

范围攻击，范围内所有敌兵都会被打中；八双一次消耗八枚手里剑，未启动杀阵时八双时威力与单发手里剑威力相同，启动杀阵后受杀阵倍率影响；秀真和守恒的八双也附带定身麻痹效果，时间维持较长，敌人受到攻击(包括足踢)或死亡都会中断效果

无效

秀真和守恒的手里剑对机械化单位是无效的

掉落

杀忍者系敌兵后会随机掉落一些手里剑，若未出现杀阵画面，则每个兵最多掉落一把手里剑，若出现杀阵画面，则每个兵可能掉落一把，也可能掉落三把手里剑
