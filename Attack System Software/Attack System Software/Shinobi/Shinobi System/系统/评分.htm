评分

KILL

TATE POINT

杀阵得分，启动杀阵即有分，得分由杀阵结束或中断时字数n决定，为30n2

BOSS POINT

BOSS战得分，此关BOSS战满分除以BOSS受到刀攻击判定次数所得商即为得分(除数为零时得分为0)；若BOSS战使用忍术则此项得分为0；A关BOSS 20000分满分(2A 40000分满分)，B关BOSS 30000分满分，STAGE EX 20000分满分

MAKIMONO

过关后所剩忍卷得分，每个忍卷2000分；每关共3个忍卷(5B 6B 7B 各1个,EX2个)

NO DAMAGE

整关(关卡战和BOSS战)都没受过伤(被恶食吸阳魂不算受伤)可得10000分
