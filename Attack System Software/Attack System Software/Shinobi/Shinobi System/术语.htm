Shinobi System

作者：J

术语

S忍
指Shinobi，也叫忍或男忍，与女忍(Kunoichi/Nightshade)区别时用的简称

邪道
利用Shinobi系统漏洞(bug)

恶食
指主人公的刀或刀的能力，有“妖刀恶食”之说

恶食觉醒
源于剧情，主人公秀真在第一大关(1A 1B)无法使用恶食的能力

杀阵
用刀杀死敌人后杀阵会启动，屏幕右上角的字会燃烧

中断
杀阵过程中指定时间内没有继续杀敌延续杀阵的效果，有故意中断的用法

三面倍率

攻击敌兵的正面、侧面和背面可能会带给敌人不同的杀伤力，多数情况下三面威力正面最小，侧面居中，背面最大

一刀斩
指BOSS战过程中BOSS累计只受到一次刀的攻击判定，从而能在过关后BOSS POINT项得满分

最速
用最短的时间过关

全杀
过关评价中 KILL100%，占有KILL的敌兵死亡就能得到KILL分，不是自己亲手杀的也能得KILL分

无伤
过关评价中 NO DAMAGE 得分，简称ND

TATE
就是杀阵

满分
指过关后得到最高分，但不允许包含重复得分

壁斩
人物处于攀爬状态时用刀砍

蓄斩
蓄力斩，有时也用“蓄”简称

变向
指变向蓄力斩

串斩
指一个蓄力斩的过程中杀死不止一个敌人

补魂
魂不够启动蓄力斩时杀一定数量的敌人后中断杀阵，以吸取足够启动蓄力斩的魂量

八双
也叫八方手里剑，是“恶食八双”的简称

盲标
不依靠锁定功能用手里剑打中目标

禁标
指不使用手里剑攻击敌人，杀阵过程中不使用手里剑破封印(用刀)

DASH
瞬移，一种特殊移动方式，也叫“冲”、“残影”、“残像”，常用"D"简称

兜后
指结合锁定和方向DASH，从而达到绕到敌人背后的效果

JD
J是Jump的简称，就是跳，J和D快速连接使用，也被称之为“长DASH”

DJ
D和J在落差边缘使用，从而得到更远的移动距离，也被称为“大跳”、“超跳”、“超级跳”、“瞬跳”，属于一种bug

消招
同格斗游戏中的消招，指一招还未结束就直接使出下一招

狂刀
高频率出刀技巧

狂标
高频率出标技巧

无影刀
在刀于刀的连续使用中，使用DASH消招，但不出现由DASH消招而出现的残像
